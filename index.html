WonderWorld + LGM Integration

A New Frontier in 3D Scene Generation

One-line Summary: Enhancing WonderWorld with multi-view reasoning and Gaussian modeling for better 3D scenes.

Motivation

While WonderWorld generates interactive 3D scenes from single images, it struggles with generating complete object backsides and accurate multi-view consistency. Integrating LGM and Gaussian representations addresses these gaps, offering higher fidelity and richer reconstructions.

Method

Our pipeline consists of multiple stages: starting from a single input image, we estimate depth, generate layered views via FLAGS, reconstruct objects through ImageDream, and represent the 3D scene using a 4D Gaussian model for real-time novel view synthesis.
